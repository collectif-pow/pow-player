# Admin dashboard for the pow-player

## Install

Clone then `npm i`.

## Build

`npm run build`

## Run

`npm run server`

## Dev

-   `npm start` for the frontend
-   `npm run server:dev` for the server

## Run it forever

Use http://pm2.keymetrics.io/ to run at startup http://pm2.keymetrics.io/docs/usage/startup/ .
